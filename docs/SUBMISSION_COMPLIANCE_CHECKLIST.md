# OpenAI Build Week Submission Checklist

This document summarizes the requirements checked for the SecHub Academy submission.

## Project and Category
- Platform category is Education.
- Project integrates Codex and GPT-5.6 as documented.
- Pre-existing codebase is separated from the work done during the hackathon window.
- The hosted application and production API health URLs are verified and active.

## Repository and Documentation
- The public repository URL is set to the correct branch.
- The root repository contains a LICENSE file.
- The README provides step-by-step setup guides for development and production.
- Third-party dependency licenses and notices are documented.

## Submission Fields and Media
- The project description and title are written in English.
- The demo video is under three minutes and contains narration explaining the project.
- The demo video does not contain unauthorized third-party marks or copyrighted background music.
- The Codex Session ID for the primary development tasks is ready for Devpost.
