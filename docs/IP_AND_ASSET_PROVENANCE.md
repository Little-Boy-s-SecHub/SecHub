# Intellectual Property and Asset Register

This document registers the origin of code, diagrams, fonts, and assets used in SecHub Academy.

## Source Code Origin
- Next.js frontend: Developed within the SecHub repository under frontend/.
- Spring Boot backend: Developed within the SecHub repository under backend/.
- Lab templates: Created by the project team, located in the backend/ directory.
- Build/Deployment scripts: GitHub Actions workflow files and deployment setups.

## Image and Diagram Origin
- Mascot and Favicon (logo.jpg, icon.jpg): Created specifically for SecHub.
- Project architecture diagrams (sechub-system-design.png, sechub-system-architecture.png): Created by the project team to document the system.
- Unused temporary images (frontend/public/game/): Unused assets included in the original project template.

## Content and Datasets
- Security lesson content: Loaded from the separate content repository (Little-Boy-s-SecHub/Data).
- Practice question cards: Generated at runtime via OpenAI API requests.
- Seeded avatars: Generated using the free DiceBear Avataaars library.

## Fonts
- JetBrains Mono: Loaded from Google Fonts under the SIL Open Font License 1.1.
